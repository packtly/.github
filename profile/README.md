# Packtly

Repository management and build infrastructure for Debian and embedded Linux systems.

Packtly focuses on reproducible package pipelines built around Aptly,
Debian repositories, containerized CI/CD and automated deployments.

## Projects

| Repository | Description |
|---|---|
| `packtly-builder` | Package and image build orchestration |
| `packtly-infra` | Deployment and infrastructure automation |

## Features

- Aptly-based repository management
- Debian package publishing
- Reproducible build workflows
- Snapshot and release automation
- Container-native CI/CD
- Embedded Linux deployment tooling

## Goals

- Simplify Debian repository management
- Automate package publishing pipelines
- Provide reproducible embedded Linux workflows
- Integrate build and deployment infrastructure
